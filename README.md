# coriander_app

**Firebaseと連携するFlutterブックリストアプリ**

## 仕様
- Firebaseとデータベース連携
- FirebaseAuthでサインアップ、サインイン
- 画像をFirebaseストレージへアップロード
- アップロードした画像URLをダウンロード

などを行う

## 機能
- ブックリストに、画像とタイトル情報を持つデータを追加できる
- データを編集、削除できる
